## http协议的组成部分

> http数据包 =  Http Headers + Http Body (可以没有)

HTTP 头字段根据实际用途被分为以下 4 种类型：

+ 通用头字段(General Header Fields)
+ 请求头字段(Request Header Fields)
+ 响应头字段(Response Header Fields)
+ 实体头字段(Entity Header Fields)
